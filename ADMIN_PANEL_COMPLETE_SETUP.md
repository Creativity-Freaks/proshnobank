# Admin Panel - সম্পূর্ণ রিডিজাইন এবং সেটআপ গাইড

## সারসংক্ষেপ

Admin Panel সম্পূর্ণরূপে redesigned এবং rebuild করা হয়েছে। এখন এটি:
- Modern tabbed interface সহ
- সব ধরনের management features সহ
- Frontend এর সাথে perfectly synchronized
- Mobile-responsive এবং user-friendly

## ইনস্টলেশন এবং অ্যাক্সেস

```
URL:      http://localhost:8080/admin/login
```

## প্রধান ড্যাশবোর্ড

URL: `http://localhost:8080/admin`

### ডাশবোর্ডের 6টি ট্যাব:

#### 1. 📚 ক্যাটেগরি (Categories)
- সব পরীক্ষার ক্যাটেগরি পরিচালনা করুন
- নতুন ক্যাটেগরি যোগ করুন
- বিদ্যমান ক্যাটেগরি সম্পাদনা/মুছুন
- প্রতিটি ক্যাটেগরিতে বিষয় সংখ্যা দেখুন

**URL:** `/admin/categories`

**বৈশিষ্ট্য:**
- Card-based grid layout
- Add/Edit/Delete dialogs
- Subject count display
- Responsive design

**কীভাবে নতুন ক্যাটেগরি যোগ করবেন:**
1. "নতুন ক্যাটেগরি যোগ করুন" বাটনে ক্লিক
2. ক্যাটেগরি নাম লিখুন
3. ID (ঐচ্ছিক) লিখুন
4. সংরক্ষণ করুন

---

#### 2. 📖 বিষয় (Subjects)
- নির্বাচিত ক্যাটেগরির সব বিষয় দেখুন
- নতুন বিষয় যোগ করুন
- বিষয় সম্পাদনা/মুছুন
- প্রতিটি বিষয়ের টপিক সংখ্যা দেখুন

**URL:** `/admin/subjects`

**কর্মপ্রবাহ:**
1. ড্রপডাউন থেকে ক্যাটেগরি নির্বাচন করুন
2. "নতুন বিষয় যোগ করুন" বাটনে ক্লিক করুন
3. বিষয় তথ্য পূরণ করুন
4. সংরক্ষণ করুন

**উদাহরণ:**
- বাংলা ১ম পত্র
- ইংরেজি ১ম পত্র
- গণিত (প্রথম পত্র)
- পদার্থবিজ্ঞান

---

#### 3. 📝 অধ্যায় (Chapters)
- নির্বাচিত বিষয়ের সব অধ্যায় দেখুন
- অধ্যায় expand/collapse করুন বিস্তারিত দেখতে
- নতুন অধ্যায় যোগ করুন
- টপিক যোগ/সম্পাদনা/মুছুন

**URL:** `/admin/chapters` অথবা `/admin/batches`

**বৈশিষ্ট্য:**
- Accordion-style expandable chapters
- Topic management within chapters
- Chapter numbering (অধ্যায়-०१, অধ্যায়-०२, ইত্যাদি)
- Topics display

**নতুন অধ্যায় যোগ করা:**
1. বিষয় নির্বাচন করুন
2. "নতুন অধ্যায় যোগ করুন" ক্লিক করুন
3. পূরণ করুন:
   - অধ্যায় নম্বর (1, 2, 3...)
   - অধ্যায় নাম (গদ্য, কবিতা, নাটক, ইত্যাদি)
   - বর্ণনা (ঐচ্ছিক)
   - টপিকসমূহ (কমা দিয়ে আলাদা করুন)
4. সংরক্ষণ করুন

**উদাহরণ - বাংলা ১ম পত্র:**
```
অধ্যায়-०१: গদ্য
  টপিকস: সাহিত্য, ব্যাকরণ, রচনা

অধ্যায়-०२: কবিতা
  টপিকস: প্রাচীন কবিতা, আধুনিক কবিতা

অধ্যায়-०३: নাটক
  টপিকস: নাটকের উপাদান, দৃশ্যকল্প
```

---

#### 4. ❓ প্রশ্ন (Questions)
- সব প্রশ্ন পরিচালনা করুন
- প্রশ্ন খুঁজুন (সার্চ)
- নতুন প্রশ্ন যোগ করুন
- প্রশ্ন সম্পাদনা/মুছুন

**URL:** `/admin/questions`

**নতুন প্রশ্ন যোগ করা:**
1. "নতুন প্রশ্ন যোগ করুন" বাটনে ক্লিক করুন
2. ফর্মে পূরণ করুন:
   - বিষয় নির্বাচন
   - অধ্যায় নির্বাচন
   - টপিক নির্বাচন
   - কঠিনতা (সহজ/মাঝারি/কঠিন)
   - প্রশ্ন টেক্সট
   - 4টি অপশন
   - সঠিক অপশন নির্বাচন
   - ব্যাখ্যা (ঐচ্ছিক)
3. সংরক্ষণ করুন

---

#### 5. 👥 ব্যবহারকারী (Users)
- সব ব্যবহারকারী পরিচালনা করুন
- ব্যবহারকারী পরিসংখ্যান দেখুন
- নতুন ব্যবহারকারী যোগ করুন
- ব্যবহারকারী রোল পরিবর্তন করুন

**URL:** `/admin/users`

**পরিসংখ্যান:**
- মোট ব্যবহারকারী
- সক্রিয় ছাত্র
- শিক্ষক এবং অ্যাডমিন

---

#### 6. 📊 বিশ্লেষণ (Analytics)
- সিস্টেম পরিসংখ্যান দেখুন
- প্রশ্ন এবং পরীক্ষা সংখ্যা
- গড় স্কোর এবং পারফরম্যান্স
- সাম্প্রতিক কার্যকলাপ

**URL:** `/admin/analytics`

**মেট্রিক্স:**
- মোট প্রশ্ন: সিস্টেমে মোট প্রশ্নের সংখ্যা
- মোট পরীক্ষা: সৃষ্ট এবং সম্পন্ন পরীক্ষা
- মোট প্রচেষ্টা: সব ছাত্রদের পরীক্ষার প্রচেষ্টা
- গড় স্কোর: সব ছাত্রদের গড় স্কোর

## সাধারণ কাজগুলি

### নতুন পরীক্ষা সেটআপ করা

1. **ক্যাটেগরি তৈরি করুন** (যদি না থাকে)
   - Categories ট্যাব যান
   - "নতুন ক্যাটেগরি যোগ করুন" ক্লিক করুন
   - ক্যাটেগরি নাম লিখুন
   - সংরক্ষণ করুন

2. **বিষয় যোগ করুন**
   - Subjects ট্যাব যান
   - ক্যাটেগরি নির্বাচন করুন
   - "নতুন বিষয় যোগ করুন" ক্লিক করুন
   - বিষয় তথ্য পূরণ করুন
   - সংরক্ষণ করুন

3. **অধ্যায় এবং টপিক যোগ করুন**
   - Chapters ট্যাব যান
   - বিষয় নির্বাচন করুন
   - "নতুন অধ্যায় যোগ করুন" ক্লিক করুন
   - অধ্যায় বিস্তারিত পূরণ করুন
   - সংরক্ষণ করুন

4. **প্রশ্ন যোগ করুন**
   - Questions ট্যাব যান
   - "নতুন প্রশ্ন যোগ করুন" ক্লিক করুন
   - প্রশ্ন বিস্তারিত পূরণ করুন
   - সংরক্ষণ করুন

5. **ফ্রন্টএন্ডে পরীক্ষা তৈরি করুন**
   - অ্যাপে যান
   - Exam Setup এ যান
   - আপনার ক্যাটেগরি নির্বাচন করুন
   - বিষয় এবং টপিক নির্বাচন করুন
   - পরীক্ষা শুরু করুন

- ইমেইল VITE_ADMIN_EMAILS এ নেই

**সমাধান:**
- ইমেইল যাচাই করুন: `info.proshnobank@gmail.com`
- পাসওয়ার্ড যাচাই করুন: `admin123`
- ক্যাপস লক চেক করুন
- VITE_ADMIN_EMAILS environment variable চেক করুন

**সমাধান:**

**সম্ভাব্য কারণ:**
- Admin account দিয়ে লগইন করেননি
- ইমেইল VITE_ADMIN_EMAILS এ নেই

**সমাধান:**
- Admin ইমেইলে লগইন করুন: `info.proshnobank@gmail.com`
- VITE_ADMIN_EMAILS environment variable চেক করুন
